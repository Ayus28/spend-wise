# UI/UX Plan -  AI Spend Aduitor

## Page Layout
1. **Header**: Logo (left) and GitHub Repo Link (right).
2. **Hero**: Catchy headline about savings money on AI.
3. **Main Dashboard**:
   - Left Side: List of AI Tools with checkboxes and monthly price tags.
   - Right Side: Dynamic "Savings Caluclator" card.
4. **Footer**: "Built for the 5-Day AI Hackaton".

## Color palette (Suggested)
- **Primary**: Drak Blue / Slate (Professional look)
- **Accent**: Emerald Green (for "Savings" and "Money" vibes)
- **Warning**: Amber (For "Overlapping Tools" alerts)

##  Interactions
- When a user checks a tool, the "Total Cost" Should update insttanly.
- If two redundants tool ( e.g. , Cursor and Copilot ) are selected, a warning icon should appear.
