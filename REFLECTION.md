# Reflection - AI Spend Auditor

### 1. What was the most challenging part?
The most challenging part was managing the application state using React's `useState` while ensuring the "Budget Alert" logic reacted instantly to data changes. Also, configuring the root directory in Vercel to host a sub-folder project was a great learning experience.

### 2. What would I do differently next time?
If I were to start over, I would use a backend like Supabase to persist user data. Currently, the data resets on refresh, but a database would make it a professional-grade auditing tool.

### 3. Key Learnings
- **Next.js 14**: Mastering the App Router and folder structure.
- **Tailwind CSS**: Implementing conditional styling (Red/Green modes) based on logic.
- **Deployment**: Learning how to link GitHub with Vercel for continuous integration.
