# AI Prompts Used - AI Spend Auditor

I used AI to assist in the architectural planning and debugging of this project. Below are the key prompts used:

### 1. UI & Project Structure
- "Create a professional dashboard UI using Next.js 14 and Tailwind CSS for an AI Spend Auditor tool. The dashboard should show a list of tools, their monthly costs, and an overall optimization score."

### 2. State Management Logic
- "Help me write a React function to add and delete objects from an array of tools. The total cost should update automatically whenever a tool is added or removed."

### 3. Conditional Styling (Budget Alert)
- "I want my dashboard to change its theme color to red if the total monthly spend exceeds $60. Provide a Tailwind CSS solution for dynamic class binding based on the total cost."

### 4. Deployment Troubleshooting
- "My Next.js project is in a sub-folder named 'ai-spend-audit'. How should I configure the Root Directory on Vercel to make sure it deploys correctly from GitHub?"
