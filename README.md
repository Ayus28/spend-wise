### ✨ Features
- **Real-time Tracking**: Add and manage AI tool costs dynamically.
- **AI Budget Alerts**: Visual indicators (Red/Green) when spending exceeds $60.
- **Optimization Score**: Smart scoring based on monthly budget utilization.
- **Responsive Design**: Built with Next.js 14 and Tailwind CSS for a seamless experience.

### 🛠️ Tech Stack
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS v4
- **Deployment**: Vercel
- **Version Control**: Git & GitHub
